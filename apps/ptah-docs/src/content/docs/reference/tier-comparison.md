## Notes

- **Trial** grants full Pro functionality for 14 days. No card required. Reverts to Free when the trial ends unless you activate a Pro key.
- **Free** is permanent and non-expiring. The provider list is the same across tiers — you are not locked into a specific model behind a paywall.
- **Pro** is per-seat. Volume discounts kick in starting at 5 seats.

<Aside type="tip">
Start a trial any time from **Settings → License → Start trial**. You can also upgrade directly to Pro without trialling first.
</Aside>
