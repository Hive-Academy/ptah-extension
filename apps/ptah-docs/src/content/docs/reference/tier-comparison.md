Ptah has two tiers. **Community** is free, open source, and includes every local capability with no gating. **Ptah Builders** is a paid membership that layers on hosted and community perks — it does not unlock any local feature.

| Capability                                                           | Community | Ptah Builders |
| -------------------------------------------------------------------- | :-------: | :-----------: |
| Unlimited chat sessions                                              |    Yes    |      Yes      |
| All supported providers (Claude, Copilot, Codex, Ollama, OpenRouter) |    Yes    |      Yes      |
| Bring-your-own API keys                                              |    Yes    |      Yes      |
| Persistent memory (BM25 + vector) & codebase indexing                |    Yes    |      Yes      |
| Auto-learning Skills curator                                         |    Yes    |      Yes      |
| Sub-agent orchestration (architect/backend/frontend/tester/reviewer) |    Yes    |      Yes      |

| Cron scheduler                                                       |    Yes    |      Yes      |
| Messaging gateways (Telegram / Discord / Slack + voice)              |    Yes    |      Yes      |
| Approval relay for every diff                                        |    Yes    |      Yes      |
| Browser automation (navigate, network monitoring, GIF recording)     |    Yes    |      Yes      |
| Code execution                                                       |    Yes    |      Yes      |
| MCP driving & built-in MCP server                                    |    Yes    |      Yes      |
| Agent sync to the headless CLI                                       |    Yes    |      Yes      |

| **Community access**                                                 |     —     |      Yes      |

## Notes

- **Community** is free and fully open source. Every local capability — chat, agents, orchestration, memory, skills, cron, messaging gateway, browser automation, code execution, MCP, and all providers — is included with no gating. There is nothing to activate and no feature held behind a paywall on your own machine.
- **Ptah Builders** is a paid membership ($29/mo or $290/yr display pricing, billed via Paddle). It adds hosted and community perks — hosted gateway, priority support, early access, and community access. It does **not** unlock local features, because every local feature already ships in Community.
