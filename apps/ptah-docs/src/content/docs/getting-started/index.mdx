Get up and running with Ptah in minutes. Ptah is an AI dev team that ships production SaaS — a desktop app whose staffed agents learn your architecture through persistent memory and deliver features around the clock.

This section walks you through the three steps every new user takes: downloading and installing the app, launching it for the first time, and signing in with a license or API keys.

## What you'll need

- A supported operating system (Windows 10+, macOS 11+, or a recent Linux distribution)
- A project folder you want to work on
- At least one AI provider you can authenticate with — a Claude, Copilot, Codex, Ollama, or OpenRouter account, or a local model

Ready? Head to the [installation guide](/getting-started/installation/) to download Ptah.
