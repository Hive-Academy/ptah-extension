# Script 01 — Open Source Announcement

**Mode:** `talking-head`
**Target duration:** 90–120s
**Synopsis:** The founder explains that Ptah is now fully open source, why the license gate came off, what's free, and what still costs money.

---

Ptah is fully open source now. Every local feature is free, permanently — the multi-agent canvas, persistent memory, skills, scheduled agents, the messaging gateway, and all three runtimes: the VS Code extension, the Electron desktop app, and the CLI.

[lower-third: title="Abdallah Khalil", subtitle="Founder, Ptah"]
[keyword: open source]

Here's the reasoning. We used to gate those features behind a license. But this space already has strong free alternatives — Hermes Agent has around a hundred and ten thousand GitHub stars, OpenClaw is past three hundred and seventy thousand, and both already cover memory, skills, subagents, cron, and messaging. Charging for capabilities that free projects already do well stopped making sense. There's a second reason too: the product gets better with more people using it — more real workflows, more edge cases found, more skills contributed back.

[keyword: free alternatives]

So here's what's free, starting today. Run three or more agents on one codebase in the orchestra canvas.

[b-roll: canvas-orchestra — while saying "orchestra canvas"]

Give Ptah a persistent memory that survives every session.

[b-roll: memory-recall — while saying "persistent memory"]

Let it learn its own skills from finished work. Schedule agents to run overnight with cron. Drive them from Telegram, Discord, or Slack.

[b-roll: gateway-tour — while saying "Telegram, Discord, or Slack"]

And use whichever runtime fits how you work: VS Code, Electron, or headless CLI. None of it needs a license key.

[keyword: no license key]

What we do charge for is Ptah Builders: a membership built around one course — build a production SaaS with Ptah — where you keep the reference codebase and the exact agent setup that built it.

[keyword: Builders]
[layout: screen-full-with-bubble]

If you want to follow along, the waitlist is at ptah.live.
