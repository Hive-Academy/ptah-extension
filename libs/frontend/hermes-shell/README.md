# @ptah-extension/hermes-shell

Angular library that provides the canonical Hermes hub shell — a four-tab
container (Memory, Skills, Schedules, Messaging) for the agentic platform
features in the Ptah webview. Tab content components are supplied by the
`*-ui` feature libraries (`memory-curator-ui`, `skill-synthesis-ui`,
`cron-scheduler-ui`, `messaging-gateway-ui`).
